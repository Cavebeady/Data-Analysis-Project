## Introduction
This project contains a data analysis system built to process 12 months of transaction data. It turns the data into excel dashboards and text reports using Python, Numpy and Pandas. 
# Key Features
* Data cleaning (removal of null values and handling outliers)
* Numpy statistical analysis ( statistical operations, broadcasting and outlier detection)
* Data manipulation with pandas (merging of datasets, extraction of columns and table aggregations)
* Generation of .txt summaries and multi-sheet .xlsx report
